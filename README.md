# Julia-Mini-Projects

Mini projets pour apprendre le langage Julia

- Story-Generator.jl : génère des petites histoires aléatoires
- Guess-the-Number.jl : jeu simple pour deviner un nombre

## Comment lancer
1. Ouvrir un terminal (PowerShell) dans le dossier du projet.
2. Lancer un script avec Julia :
  - julia Story-Generator.jl
  - julia Guess-the-Number.jl

## Prérequis
- Julia installée (https://julialang.org)
